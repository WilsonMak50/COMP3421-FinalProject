Bootstrap Navbar with Two Lines

This is an example of a Bootstrap navbar with two lines. The first line is larger and bolder, while the second line is smaller and lighter. To achieve this, we've added two span elements within the navbar-brand anchor tag, and applied the d-block class to each one. This makes each span element take up its own line within the navbar-brand, creating the two-line effect.
